# Capstone-Project-1   
## PROFIT ANALYSIS USING LINEAR REGRESSION WITH POWER BI DASHBOARD
### OBJECTIVE:
    Developed an interactive Profit Analysis Dashboard using Power BI to analyze sales, revenue, cost, and profit performance across different categories, regions, and time periods. The dashboard helps in identifying high-performing products and areas with low profitability to support data-driven business decisions.

### TOOLS USED:
    Power BI – Data visualization & dashboard creation
    Microsoft Excel / CSV Dataset – Data source
    Power Query – Data cleaning & transformation
    DAX (Data Analysis Expressions) – Calculated measures (Profit, Profit %, KPIs)

### INSIGHTS:
    Identified high-profit and low-profit product categories.
    Analyzed regional sales performance to determine top-performing locations.
    Observed trends in monthly and yearly profit growth.
